# Framer Motion Project 1

A personal portfolio website built to present my skills, projects, and contact details in a modern, animated single-page layout.

## Overview

This project is designed to give visitors a quick but polished view of who I am and what I build. It includes:

- A landing hero with name, role, and call-to-action buttons
- A technology stack section
- An about section with a short personal journey
- A scrolling logo ticker
- A featured projects section
- A contact section with phone, email, office details, and a map
- A footer with social links

## Preview

- https://framer-motion-project-1.netlify.app

## Built With

- `Next.js`
- `React`
- `TypeScript`
- `Tailwind CSS`
- `Framer Motion`
- `React Three Fiber`
- `React Icons`

## Features

- Responsive design for desktop and mobile
- Animated backgrounds and transitions
- Smooth scrolling navigation
- Interactive project cards
- Modern dark theme with purple accents

## Project Structure

- `app/page.tsx` renders the full page
- `components/Navbar.tsx` handles navigation
- `components/Hero.tsx` contains the hero section
- `components/Stack.tsx` shows the tech stack
- `components/About.tsx` presents the about section
- `components/LogoTicker.tsx` shows partner/brand logos
- `components/Portfolio.tsx` displays featured projects
- `components/Contact.tsx` provides contact information
- `components/Footer.tsx` contains social links

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` starts the development server
- `npm run build` builds the app for production
- `npm run start` runs the production server
- `npm run lint` checks the code with ESLint

## Notes

- Some content in the project is still placeholder text and can be updated with real personal details.
- Images used throughout the site are stored in `public/images`.
